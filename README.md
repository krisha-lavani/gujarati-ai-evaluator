# Gujarati AI Response Evaluator

A benchmarking system to evaluate Gujarati AI responses across multiple LLMs.

## Models Compared
- ChatGPT
- Grok
- Gemini

## Features
- Gujarati language scoring
- Fluency scoring
- Safety scoring
- Accuracy scoring
- Hallucination detection
- Weighted model ranking

## Tech Stack
- Python
- Pandas
- Matplotlib
- Streamlit

## Results
Compared 3 LLMs across 25 prompts.

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
